# CODEGENERATOR
REST API in Java con Spring Boot per generare automaticamente codice C# pronto per Visual Studio. Include endpoint /generate per creare classi e snippet, documentazione Swagger/OpenAPI integrata e struttura modulare estendibile a interi progetti. Ideale per dimostrare competenze avanzate in architettura REST.
# CodeGenerator
